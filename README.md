# UDP Real-Time Multiplayer Game Engine

A simple real-time multiplayer networking engine built using UDP sockets in Python.

## Features

- UDP based multiplayer networking
- Real-time player movement
- Server state synchronization
- Latency measurement (Ping/Pong)
- Consistent player colors
- Multiple clients supported

## Technologies Used

- Python
- UDP Socket Programming
- Pygame

## How To Run

### 1 Start the server

python server.py

### 2 Start clients

python client.py

python client.py

## Controls

W - Move Up  
A - Move Left  
S - Move Down  
D - Move Right

## Demo

Each player appears as a colored circle with their player number.

Latency is displayed in the top corner.
